Advanced: Implementing the Convesation Meme

1. Overview

The purpose of this tutorial is to show how one of the fundamental memes is implemented. This meme is rather complex because of the way the conversation system works in NeverwinterNights.

This tutorial will ignore the process of adding the meme to a library and concentrate on how local variables are used and inherited.

2. A Review of Variable Inheritance

The MT scripts support variable inheritance. This means that variables on an object can be redirected to a parent object.
